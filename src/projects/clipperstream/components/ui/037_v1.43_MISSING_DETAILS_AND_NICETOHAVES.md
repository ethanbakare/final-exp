### CRITICAL Missing Detail #1: Queue-Wide Circuit Breaker Behavior

**From your clarification**:
> "Once it fails once at the start of handling pending clips, it would use OpenAI for every other member in the queue. So it's not going to fail every time - it's only going to fail once at the very start."

**What v1.39 Currently Says**:
- DNS error → Switch to Whisper immediately ✅
- API down (502/503/504) → Retry twice, then switch ✅

**What's MISSING**: The queue behavior detail!

**CORRECT Queue Behavior** (Should be in v1.39):

```
User goes offline → Records 5 clips → Goes online (VPN on)
  ↓
Auto-retry starts processing queue SEQUENTIALLY:

Clip 1 (First in queue):
  ├─ Try Deepgram → DNS error (ENOTFOUND)
  ├─ Circuit breaker opens IMMEDIATELY
  └─ Switch to Whisper → Success ✅

Clip 2 (Second in queue):
  ├─ Circuit breaker is OPEN
  ├─ Skip Deepgram entirely
  └─ Use Whisper directly → Success ✅

Clip 3, 4, 5 (Remaining):
  ├─ Circuit breaker STILL OPEN
  ├─ Skip Deepgram for ALL remaining clips
  └─ Use Whisper directly → Success ✅

Result: Only Clip 1 tries Deepgram. Rest use Whisper immediately.
Time saved: ~4 seconds (no wasted DNS failures)
```

**Key Point**: Circuit breaker state persists ACROSS the queue processing session. Once it opens for Clip 1, it stays open for Clips 2-5.

**Why this matters**:
- ✅ No wasted time retrying Deepgram for every clip
- ✅ Queue processes faster
- ✅ Only fails once, then learns

**This should be added to v1.39 under the VPN Context section.**

**CORRECT Behavior by Queue Type**:

**Queue 1: Auto-Retry (Background)**
```
Processing pending clips from offline→online
  ↓
Clip 1 fails with DNS → Circuit opens
  ↓
Circuit stays OPEN for entire queue processing session
  ↓
All remaining clips use Whisper
```

**Queue 2: New Recordings (Foreground)**
```
User records NEW clip while queue is processing
  ↓
Circuit is OPEN (from Queue 1)
  ↓

  ↓
```

**Solution**: Circuit breaker behavior differs by context:
- **Auto-retry context**: Circuit opens immediately for DNS, stays open for queue
- **New recording context**: Try Deepgram first (fresh connection might work)

```typescript
import { useEffect, useCallback } from 'react';
import { Audio } from 'expo-av';

/**
 * Handles audio interruptions (phone calls, alarms, etc.)
 * Auto-triggers callback when interruption begins
 */
export function useAudioInterruption(
  onInterruption: () => void,
  isRecording: boolean
) {
  const handleInterruption = useCallback((event: Audio.AudioInterruptionEvent) => {
    if (event.type === Audio.INTERRUPTION_TYPE_BEGAN && isRecording) {
      console.log('[Audio] Interruption detected - auto-finishing recording');
      onInterruption();
    }
  }, [onInterruption, isRecording]);
  
  useEffect(() => {
    // Set up audio session
    Audio.setAudioModeAsync({
      allowsRecordingIOS: true,
      playsInSilentModeIOS: true,
      staysActiveInBackground: true,
      shouldDuckAndroid: true,
      playThroughEarpieceAndroid: false,
      interruptionModeIOS: Audio.INTERRUPTION_MODE_IOS_DO_NOT_MIX,
      interruptionModeAndroid: Audio.INTERRUPTION_MODE_ANDROID_DO_NOT_MIX,
    });
    
    // Listen for interruptions
    const subscription = Audio.setOnAudioInterruptionEventListener(handleInterruption);
    
    return () => subscription.remove();
  }, [handleInterruption]);
}
```

```typescript
import AsyncStorage from '@react-native-async-storage/async-storage';

const STORAGE_KEY = 'circuit-breaker-state';

interface CircuitBreakerState {
  state: 'CLOSED' | 'OPEN';
  failureCount: number;
  lastFailureTime: number;
}

class TranscriptionCircuitBreaker {
  private state: 'CLOSED' | 'OPEN' = 'CLOSED';
  private failureCount: number = 0;
  private lastFailureTime: number = 0;
  private initialized: boolean = false;
  
  /**
   * Initialize circuit breaker (call on app startup)
   */
  async init(): Promise<void> {
    if (this.initialized) return;
    
    try {
      const saved = await AsyncStorage.getItem(STORAGE_KEY);
      if (saved) {
        const data: CircuitBreakerState = JSON.parse(saved);
        this.state = data.state;
        this.failureCount = data.failureCount;
        this.lastFailureTime = data.lastFailureTime;
        
        console.log('[CircuitBreaker] Restored state:', data);
        
        // Auto-reset if last failure was > 5 minutes ago
        const timeSinceFailure = Date.now() - this.lastFailureTime;
        if (timeSinceFailure > 5 * 60 * 1000) {
          console.log('[CircuitBreaker] State expired - resetting');
          this.reset();
        }
      }
    } catch (error) {
      console.error('[CircuitBreaker] Failed to restore state:', error);
    }
    
    this.initialized = true;
  }
  
  /**
   * Save state to AsyncStorage
   */
  private async saveState(): Promise<void> {
    try {
      const state: CircuitBreakerState = {
        state: this.state,
        failureCount: this.failureCount,
        lastFailureTime: this.lastFailureTime
      };
      
      await AsyncStorage.setItem(STORAGE_KEY, JSON.stringify(state));
    } catch (error) {
      console.error('[CircuitBreaker] Failed to save state:', error);
    }
  }
  
  /**
   * Check if Whisper should be used
   */
  shouldUseWhisper(): boolean {
    return this.state === 'OPEN';
  }
  
  /**
   * Record a failure
   */
  async recordFailure(error: Error): Promise<void> {
    const isDNSError = error.message.includes('ENOTFOUND') ||
                       error.message.includes('DNS');
    
    const isAPIDown = error.message.includes('502') ||
                      error.message.includes('503') ||
                      error.message.includes('504');
    
    if (isDNSError || isAPIDown) {
      this.failureCount++;
      this.lastFailureTime = Date.now();
      
      if (isDNSError || this.failureCount >= 2) {
        console.log('[CircuitBreaker] Opening circuit');
        this.state = 'OPEN';
      }
      
      await this.saveState();
    }
  }
  
  /**
   * Record a success
   */
  async recordSuccess(): Promise<void> {
    if (this.state !== 'CLOSED') {
      console.log('[CircuitBreaker] Closing circuit (success)');
    }
    
    this.state = 'CLOSED';
    this.failureCount = 0;
    this.lastFailureTime = 0;
    
    await this.saveState();
  }
  
  /**
   * Manual reset (call after queue completes)
   */
  async reset(): Promise<void> {
    console.log('[CircuitBreaker] Manual reset');
    this.state = 'CLOSED';
    this.failureCount = 0;
    this.lastFailureTime = 0;
    
    await this.saveState();
  }
}

export const circuitBreaker = new TranscriptionCircuitBreaker();
```

```typescript
/**
 * Get all audio IDs currently in IndexedDB
 */
export async function getAllAudioIds(): Promise<string[]> {
  const db = await initDB();
  
  return new Promise((resolve, reject) => {
    const transaction = db.transaction(['audioStore'], 'readonly');
    const store = transaction.objectStore('audioStore');
    const request = store.getAllKeys();
    
    request.onsuccess = () => {
      resolve(request.result as string[]);
    };
    
    request.onerror = () => {
      reject(request.error);
    };
  });
}

/**
 * Delete audio from IndexedDB by ID
 */
export async function deleteAudio(audioId: string): Promise<void> {
  const db = await initDB();
  
  return new Promise((resolve, reject) => {
    const transaction = db.transaction(['audioStore'], 'readwrite');
    const store = transaction.objectStore('audioStore');
    const request = store.delete(audioId);
    
    request.onsuccess = () => {
      console.log('[AudioStorage] Deleted audio:', audioId);
      resolve();
    };
    
    request.onerror = () => {
      reject(request.error);
    };
  });
}
```
