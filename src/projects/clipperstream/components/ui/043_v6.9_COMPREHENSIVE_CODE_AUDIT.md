## APPENDIX A: Valid Type Reference

### ClipStatus (Canonical)
```typescript
type ClipStatus =
  | null                  // Completed (has content)
  | 'transcribing'        // HTTP transcription in progress
  | 'formatting'          // AI formatting in progress
  | 'pending-child'       // Offline, waiting to transcribe
  | 'pending-retry'       // Online, retrying after error
  | 'audio-corrupted'     // Permanent: audio retrieval failed
  | 'no-audio-detected';  // Permanent: no speech in audio
```

### LastError (Canonical)
```typescript
type LastError =
  | 'dns-block'    // VPN/DNS blocking API
  | 'api-down'     // API server error (500, 502, 504)
  | 'network'      // Network timeout/connectivity issue
  | 'validation'   // Invalid audio format/data
  | null;          // No error
```

### ClipOfflineStatus (UI Component)
```typescript
type ClipOfflineStatus =
  | 'waiting'           // Pending, not started
  | 'retry-pending'     // Failed, waiting to retry
  | 'transcribing'      // HTTP in progress
  | 'vpn-blocked'       // DNS block error
  | 'audio-corrupted'   // Permanent error
  | 'no-audio-detected'; // Permanent error
```

- ❌ `'retry'` → Use `'pending-retry'`
- ❌ `'pending'` → Use `'pending-child'` or `'pending-retry'`
