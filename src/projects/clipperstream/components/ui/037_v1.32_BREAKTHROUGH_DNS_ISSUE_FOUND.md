## Possible Root Causes

### 1. Intermittent Network Issues ⚠️ **MOST LIKELY**

**Evidence**:
- Online recording works (DNS resolved successfully)
- Minutes later, offline retry fails (DNS fails)
- Same session, same code, different result

**Possible causes**:
- Network switched (WiFi → Cellular)
- VPN connected/disconnected
- Router DNS cache expired
- ISP DNS server had temporary issue

**How to test**:
```bash
# In terminal, try multiple times
nslookup api.deepgram.com
# or
dig api.deepgram.com
# or
ping api.deepgram.com
```

If this works sometimes and fails other times → intermittent network issue.

---

### 2. DNS Caching Issues

**Theory**:
- First API call (online recording): DNS resolves, gets cached
- Cache expires
- Retry happens, DNS lookup fails because:
  - DNS server changed
  - Network changed
  - DNS cache was cleared

**Node.js behavior**:
Node.js 18+ uses undici for fetch, which has its own DNS caching behavior.

---

### 3. Network Blocking api.deepgram.com

**Possible blockers**:
- Corporate firewall
- ISP blocking
- Parental controls
- Antivirus/security software
- VPN/proxy blocking specific domains

**How to test**:

## **SOLUTION 2: Add Retry Logic with Exponential Backoff**

**Theory**: DNS lookup might be temporarily down, retry could succeed.

### Implementation

**File**: `src/projects/clipperstream/api/deepgramProvider.ts`

**Add retry helper** (before `transcribeAudio`):
```typescript
async function fetchWithRetry(
  url: string,
  options: RequestInit,
  maxRetries = 3
): Promise<Response> {
  let lastError: Error | null = null;

  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      log(`Attempt ${attempt}/${maxRetries}`);
      const response = await fetch(url, options);
      return response; // Success!
    } catch (err) {
      lastError = err as Error;

      // Check if DNS error
      if (err instanceof Error && err.message.includes('ENOTFOUND')) {
        logError(`DNS lookup failed on attempt ${attempt}`);

        // Don't retry on last attempt
        if (attempt < maxRetries) {
          // Exponential backoff: 1s, 2s, 4s
          const delay = Math.pow(2, attempt - 1) * 1000;
          logError(`Retrying in ${delay}ms...`);
          await new Promise(resolve => setTimeout(resolve, delay));
          continue;
        }
      }

      // Not a DNS error, or last attempt - throw immediately
      throw err;
    }
  }

  throw lastError || new Error('Fetch failed after retries');
}
```

**Update fetch call** (line 100):
```typescript

  'https://api.deepgram.com/v1/listen?...',

    },

```

```typescript
import fs from 'fs';
import OpenAI from 'openai';

export interface TranscriptionResult {
  transcript: string;
  duration: number;
}

export async function transcribeAudio(
  filePath: string,
  mimeType: string,
  apiKey: string
): Promise<TranscriptionResult> {
  if (!apiKey) {
    throw new Error('OpenAI API key is not configured');
  }

  const openai = new OpenAI({ apiKey });

  try {
    console.log('=== 🎯 OPENAI WHISPER API CALL ===');
    console.log('File path:', filePath);
    console.log('Using Whisper for offline retry');

    // OpenAI expects a file stream, not buffer
    const audioFile = fs.createReadStream(filePath);

    const transcription = await openai.audio.transcriptions.create({
      file: audioFile,
      model: 'whisper-1',
      response_format: 'verbose_json',
    });

    if (!transcription.text || transcription.text.trim() === '') {
      throw new Error('No speech detected in recording. Please try again and speak clearly.');
    }

    console.log('✅ Whisper transcription successful');

    return {
      transcript: transcription.text.trim(),
      duration: transcription.duration || 0,
    };
  } catch (err) {
    console.error('=== OPENAI WHISPER ERROR ===');
    console.error('Error:', err);
    console.error('=============================');

    const errorMessage = err instanceof Error ? err.message : 'Unknown error';
    throw new Error(`Transcription failed: ${errorMessage}`);
  }
}
```
